Project Overview

SignSpeak is a revolutionary Speech to Sign Language Translator that bridges communication gaps between hearing and deaf communities. The application converts spoken words into animated sign language GIFs, making communication more accessible and inclusive.

üéØ Mission

To create technology that breaks down communication barriers and promotes inclusivity for the deaf and hard-of-hearing community.

Features

Real-time Speech Recognition

Advanced speech-to-text conversion using Google Speech Recognition API

Sign Language Animation

High-quality animated GIFs showing accurate sign language gestures

Web Interface

Beautiful, responsive web interface with modern design

Desktop Application

Python-based desktop application for offline usage

System Requirements

Web Application

  • Modern web browser (Chrome, Edge, Firefox, Safari)
  • Working microphone
  • Internet connection for speech recognition
  • HTTPS connection (for production deployment)

Desktop Application

  • Python 3.7 or higher
  • OpenCV for video processing
  • Speech Recognition library
  • Working microphone and speakers
  • Internet connection

How It Works

  1. User clicks the microphone button
  2. Browser requests microphone permission
  3. Audio input is captured and processed
  4. Speech is converted to text using Web Speech API
  5. Text is matched against supported phrases
  6. Corresponding GIF animation is displayed
