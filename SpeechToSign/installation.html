The easiest way to use SignSpeak is through the web interface. No installation required!

Getting Started

  2. Navigate to the SignSpeak website
  3. Allow microphone permissions when prompted
  4. Click the microphone button and start speaking!

üêç Python Setup

Step 1: Install Python

Windows:

# Download Python from python.org
# During installation, check "Add Python to PATH"
python --version  # Verify installation

macOS:

# Using Homebrew (recommended)
brew install python3

# Or download from python.org
python3 --version  # Verify installation

Linux (Ubuntu/Debian):

sudo apt update
sudo apt install python3 python3-pip
python3 --version  # Verify installation

Step 2: Download SignSpeak

# Clone the repository
git clone https://github.com/your-repo/signspeak.git
cd signspeak

# Or download ZIP file and extract

üì¶ Installing Dependencies

Automatic Installation

# Navigate to project directory
cd signspeak

# Install all dependencies
pip install -r requirements.txt

Manual Installation

# Core dependencies
pip install opencv-python
pip install speechrecognition
pip install pyaudio
pip install numpy
pip install matplotlib

# Optional dependencies
pip install mediapipe  # For hand detection
pip install tkinter     # For GUI (usually included)

üîß Common Installation Issues

PyAudio Installation Error:

  • Windows: Download wheel from https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
  • macOS: brew install portaudio && pip install pyaudio
  • Linux: sudo apt install python3-dev portaudio19-dev && pip install pyaudio

üé§ Microphone Setup

Testing Your Microphone

# Test microphone input
python -c "
import speech_recognition as sr
r = sr.Recognizer()
with sr.Microphone() as source:
    print('Say something!')
    audio = r.listen(source)
    try:
        print('You said: ' + r.recognize_google(audio))
    except:
        print('Could not understand audio')
"

Microphone Configuration

Windows

  1. Right-click sound icon in system tray
  2. Select "Open Sound settings"
  3. Choose your microphone under "Input"
  4. Test microphone and adjust levels

macOS

  1. Open System Preferences ‚Üí Sound
  2. Click "Input" tab
  3. Select your microphone
  4. Adjust input volume

Linux

  1. Open Sound settings (varies by desktop)
  2. Or use: alsamixer command
  3. Select input device
  4. Test with: arecord -d 5 test.wav && aplay test.wav

üîß Troubleshooting

‚ùå Import Errors

Problem: ModuleNotFoundError: No module named 'cv2'

Solution:

  • Reinstall OpenCV: pip uninstall opencv-python && pip install opencv-python
  • Check Python version compatibility
  • Use virtual environment: python -m venv signspeak && source signspeak/bin/activate

üé§ Microphone Not Detected

Problem: "No microphone found" error

Solution:

  • Check microphone connections
  • Verify system permissions
  • Install PyAudio properly
  • Test with: python -m speech_recognition

üñºÔ∏è GIF Files Not Found

Problem: "File not found" for GIF animations

Solution:

  • Ensure all GIF files are in the project directory
  • Check file names match exactly (case-sensitive)
  • Run: python test_gif.py to verify files

üåê Internet Connection Issues

Problem: Speech recognition not working

Solution:

  • Check internet connection
  • Try different speech recognition service
  • Use offline speech recognition (if available)
  • Check firewall settings
